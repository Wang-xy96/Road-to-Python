5.6 人生的不同阶段


```python
age = 23
if age < 2:
    print('Baby')
elif 2 <= age < 4:
    print('Toddle')
elif 4 <= age < 13:
    print('Child')
elif 13 <= age < 20:
    print('Teenager')
elif 20 <= age < 65:
    print('Adult')
elif age <= 65:
    print('The old')
```

    Adult
